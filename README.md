# CloudAGI — Agent Credit Economy

Buy and sell unused coding agent credits. Turn wasted subscriptions into income.

## The Problem

Engineers pay $20-200/month for coding agent subscriptions (Claude Code, Cursor, Amp, Codex, Copilot) and routinely waste 50-80% of their credits. Meanwhile, others need AI compute for specific tasks but don't want monthly commitments.

## The Solution

CloudAGI is a marketplace where:
- **Sellers** list their unused coding agent credits
- **Buyers** pay per-task (build a website, create a PR, review code)
- **Routing** matches tasks to the best available agent based on strategy (cheapest, fastest, best-quality)
- **Payment** flows through x402 protocol (crypto) or Stripe (fiat)

## Why Now

- Coding agent subscriptions are exploding (Claude Code, Cursor, Amp, Codex, Copilot, Jules, Gemini CLI)
- Most users don't max out their limits — massive waste across the ecosystem
- Local model economy emerging (OpenClaw, Ollama) — sell your GPU compute too
- x402 protocol makes machine-payable APIs trivial (proven at Nevermined hackathon)

## Origin

Built during the [Nevermined hackathon](https://nevermined.app) (Mar 5-6, 2026) in SF. The hackathon proved agent-to-agent commerce works. This project extends that into a consumer-facing credit marketplace.

- Hackathon repo: [shlawgathon/CloudAGI](https://github.com/shlawgathon/CloudAGI)
- Live demo: [cloudagi.org](https://cloudagi.org)
